# Progetto Blast Locale

Script in Python per eseguire allineamenti di sequenze DNA (local alignment) tramite l'algoritmo di Smith-Waterman. 
Il software Ã¨ ottimizzato per girare su Raspberry Pi e dispone di un'interfaccia testuale (TUI).

## Installazione e Setup

1. Scaricare i file `engine.py` e `tui.py` nella stessa cartella.
2. Creare una sottocartella chiamata `genomes/`.
3. Inserire i file dei genomi (formato .fa, .fasta o .txt) dentro la cartella `genomes/`.

## Utilizzo

python3 tui.py
